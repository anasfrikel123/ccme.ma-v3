# Endpoints agents should prefer over scraping HTML:
#
#   GET /api/agent-manifest.json   — discovery + capability index
#   GET /api/services.json         — service catalogue
#   GET /api/contact.json          — contact info, hours, address
#   GET /api/cabinet.json          — cabinet profile, sectors, methodology
#   GET /api/blog.json             — blog index
#   GET /api/reviews.json          — verified client reviews
#   GET /blog/<slug>.md            — raw markdown of any blog article
#   GET /llms-full.txt             — full site text dump
#
# All endpoints return application/json or text/markdown (utf-8), permit
# cross-origin reads, and are cached at the edge for 1 hour with stale-
# while-revalidate. Authentication is not required.
